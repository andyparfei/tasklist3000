
# Taskmaster3000

## Getting started with Taskmaster3000

### 1. Clone the repo

```bash
git clone https://github.com/andyparfei/taskmaster3000
```

### 1. Run the backend using docker
```bash
cd backend
```

### 1. Run the frontend directly
```bash
cd frontend
cd myapp
npm install
npm run dev
```
Then open http://localhost:5173 in your browser. Enjoy!
